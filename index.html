404Learning

404Learning est un projet WordPress conçu avec le thème OceanWP, intégrant une gestion avancée des contenus via les taxonomies personnalisées. Le site utilise WooCommerce pour l’e-commerce, HubSpot pour le CRM et l’automatisation marketing, et est connecté à Digiforma via API pour la gestion des formations.

404Language

404Language est une plateforme d’apprentissage du français axée sur la pratique orale. Piper et Whisper tournent en local pour l’analyse et la génération de la prononciation des apprenants. La plateforme adapte les parcours aux différents niveaux d’apprenants et propose des exercices avec feedback immédiat. L’application est conteneurisée avec Docker et déployée automatiquement via GitHub Actions.
